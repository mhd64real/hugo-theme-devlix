# Devlix

A small Hugo theme for a personal site with a blog. Plain HTML, one CSS file, one tiny JS file, light and dark out of the box. The site works fully without JavaScript.

This page is the live demo. The docs live in the [Blog](/blog/): start with [Getting Started](/blog/getting-started.html).

{{< terminal >}}
$ git submodule add https://github.com/mhd64real/hugo-theme-devlix themes/devlix
$ hugo server -D
{{< /terminal >}}
